# payload-start — Структура проекта

[[000 Index|← Назад к карте]]

> [!info] Стек
> Next.js 15 + Payload CMS 3.78 + SQLite + React 19 + Tailwind v4

## Структура папок

```
src/
├── app/
│   ├── (frontend)/        ← публичный сайт
│   │   ├── page.tsx       ← главная
│   │   ├── products/      ← каталог + [id]
│   │   ├── cart/          ← корзина
│   │   ├── orders/        ← история заказов
│   │   ├── login/         ← вход клиента
│   │   ├── register/      ← регистрация
│   │   └── graphql-demo/  ← демо GraphQL
│   ├── (payload)/         ← Payload (admin + API)
│   │   ├── admin/         ← автогенерируемая UI
│   │   └── api/           ← REST + GraphQL
│   └── api/               ← кастомные роуты
│       ├── auth/          ← customer-login/logout
│       ├── checkout/
│       └── products-count/
├── collections/           ← модели данных
├── globals/               ← глобальные настройки
├── hooks/                 ← lifecycle hooks
├── access/                ← access control функции
├── components/            ← React компоненты
├── contexts/              ← React Context
└── payload.config.ts      ← главный конфиг
```

## Коллекции

| Коллекция | Auth | Особенность |
|-----------|------|-------------|
| `Users` | ✅ | Администраторы, доступ к /admin |
| `Customers` | ✅ | Клиенты, публичная регистрация |
| `Products` | — | Rich text, relationship, hooks, image |
| `Categories` | — | Простая справочная коллекция |
| `Orders` | — | Array items, email hook, access по owner |
| `Media` | — | Upload, 3 размера (thumbnail/card/hero) |

## Globals

- **SiteSettings** — название магазина, email, баннер

## Двойная аутентификация

```
Users (admin)     → Payload встроенная auth → /admin
Customers (front) → JWT в httpOnly cookie   → /api/auth/customer-login
```
