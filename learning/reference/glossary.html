React 19 · Next 15 (App Router) · Payload 3.78 — термины проекта payload-start. Живой документ, пополняется.

Component (компонент)
Функция, возвращающая JSX (описание UI). В этом проекте все компоненты — функциональные.ExtJS: аналог Ext.define/виджета, но это просто функция, без классов и config.
JSX
Синтаксис «HTML внутри JS». Компилируется в вызовы функций, создающие дерево элементов. Не строка и не DOM — описание того, что нужно отрисовать.
Server Component (RSC)
Компонент, который выполняется только на сервере. Может быть async, ходить в БД, читать файлы. В браузер уходит уже готовый результат — 0 KB JS этого компонента. По умолчанию в App Router всё — Server Component.Пример в проекте: products/page.tsx.
Client Component
Компонент с директивой 'use client' вверху файла. Выполняется в браузере, может использовать состояние, эффекты, обработчики событий (useState, onClick).Пример: CartContext.tsx, AddToCartButton.tsx.
'use client'
Директива-граница. Файл и всё, что он импортирует, становится клиентским бандлом. Ставится там, где нужна интерактивность, а не «на всякий случай».
App Router
Современная система маршрутизации Next.js, основанная на папках в src/app/. Файл page.tsx = маршрут, layout.tsx = обёртка.
Route Group (name)
Папка в скобках, например (frontend). Группирует маршруты и даёт общий layout, но не попадает в URL.
Hook (хук)
Функция вида useXxx, дающая компоненту память/побочные эффекты: useState, useReducer, useEffect, useMemo. Работает только в Client Components.
Payload Collection
Описание сущности данных (таблица + админка + API). Определяется в src/collections/.ExtJS: аналог Model + Store, но генерирует ещё и REST/GraphQL и админ-панель.
Local API
Прямой вызов Payload из серверного кода без HTTP: const payload = await getPayload({config}); await payload.find({collection:'products'}). Используется внутри Server Components.
revalidate
Экспорт из страницы, управляющий кэшем. export const revalidate = 0 = не кэшировать, рендерить заново на каждый запрос.
