# React + Next.js + Payload Resources

## Knowledge

- [React Docs — react.dev](https://react.dev/learn)
  Официальная новая документация (hooks-first, React 19). Use for: рендеринг, хуки, `useReducer`, `useMemo`, правила хуков.
- [React — Server Components (RFC / docs)](https://react.dev/reference/rsc/server-components)
  Концепция RSC. Use for: понимание server vs client components на уровне React.
- [Next.js Docs — App Router](https://nextjs.org/docs/app)
  Официальная дока Next 15. Use for: routing, layouts, data fetching, `'use client'`, route handlers, caching/`revalidate`.
- [Next.js — Server and Client Components](https://nextjs.org/docs/app/building-your-application/rendering/composition-patterns)
  Use for: паттерны композиции (когда что делать server, когда client).
- [Payload CMS Docs](https://payloadcms.com/docs)
  Use for: collections, fields, access control, hooks, Local API (`payload.find/create`), globals.
- [Payload — Local API](https://payloadcms.com/docs/local-api/overview)
  Use for: как `getPayload({config})` и `payload.find()` работают внутри Server Components.

## Wisdom (Communities)

- [Payload Discord](https://discord.com/invite/payload)
