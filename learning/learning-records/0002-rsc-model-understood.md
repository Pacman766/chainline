# Понял модель Server vs Client Components (Урок 1)

Прошёл квиз урока 1 без проблем. Различает Server vs Client Components,
понимает: RSC выполняется на сервере и может `await` данные напрямую, `'use client'` —
граница для интерактивности, секреты безопасны в Server Component, хуки запрещены без `'use client'`.

**Implications:** можно опираться на RSC-модель в следующих темах. Готов к практике —
самостоятельно создать Server Component страницу с фетчем из Payload Local API.
Следующая теория после практики: routing в App Router (layouts, route groups, dynamic `[id]`).
